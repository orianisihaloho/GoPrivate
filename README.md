## Getting Started
### Requirements
* SQL Server 2014
* XAMPP

### Usage
1. Clone this project
2. Place the project under the htdocs directory, ex: E:\xampp\htdocs\GoPrivate
3. Start the Apache on the XAMPP
4. Open browser and go to localhost:xxxx/GoPrivate
